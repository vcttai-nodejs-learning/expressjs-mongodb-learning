140. Processing Post Requests with Body Parser
- Xử lí form data với body-parser
  npm install body-parser 
- Cấu hình sử dụng body parser để lấy dữ liệu từ form:
  app.use(bodyParser.urlencoded({extended: true}));
  Giải thích:
    + bodyParser.urlencoded là để lấy dữ liệu từ form, còn nhiều mode khác như: bodyParser.json(), bodyParser.text(), ...
    + tham số extended là để chấp nhận các object lồng nhau
- Sử dụng trong callback của route:
app.post('/', function(req, res) {
    const num1 = parseInt(req.body.first_number);
    const num2 = parseInt(req.body.second_number);
    //...
})
