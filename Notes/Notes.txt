134. Create first server with Express
- npm init
- Tạo file server.js
- cài đặt express: npm install express

const app = express();

app.listen(3000, () => {
    console.log('Start server successfully!');
});

223. Relationships in MongoDB
- Có 2 cách chính:
  + Embed bên trong 1 đối tượng, vd:
    db.product.insert({
        _id: 3,
        name: "Rubber",
        price: 1.4,
        reviews: [
            {
                author: "John",
                rating: 5,
                review: "Good Robber"
            },
            {
                author: "Wilson",
                rating: 5,
                review: "Awesome Robber"
            }
        ],
    })
  + Tách collections và dùng object reference, vd:
    Product collection có 2 documents:
      {
          _id: 1,
          name: "Pen"
      }
      {
          _id: 2,
          name: "Pencil"
      }
    Order collection sẽ tham chiếu như sau:
      {
          orderNumber: 1234,
          productsOrdered: [1, 2]
      }
