226. Reading from Your Database with Mongoose
- Mongoose là 1 ODM: https://mongoosejs.com/
- Flow làm việc với mongoose:
  + require mongoose
  + mongoose.connect(db)
  + create model with mongoose.model(), hoặc mongoose.model(mongoose.Schema)
  + using model as a ORM object
  vd:
  const mongoose = require('mongoose');
  mongoose.connect('mongodb://localhost:27017/test', {useNewUrlParser: true});

  const Cat = mongoose.model('Cat', { name: String });
  const kitty = new Cat({ name: 'Zildjian' });
  kitty.save().then(() => console.log('meow'));
- Cài đặt mongoose:
  npm i mongoose
- Document:
  + Model: https://mongoosejs.com/docs/api/model.html
